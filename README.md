 Sales Performance Dashboard (Tableau Project)

## Proje Tanımı  
Bu proje, basit bir satış verisi (örneğin: ürün, kategori, miktar, fiyat) üzerinden *veri temizleme, **analiz* ve *görselleştirme* süreçlerini gösterir.  
Amaç, veriyi anlamlı hale getirip iş kararlarını destekleyecek bir dashboard oluşturmak.

##  Kullanılan Araçlar  
- Python & Pandas — veri temizleme ve ön analiz  
- Jupyter Notebook — kodlama ortamı  
- Tableau Public — etkileşimli görselleştirme ve dashboard  
- CSV formatında basit satış verisi  

##  Yapılan Analizler  
- Kategori ve ürün bazında toplam satış tutarlarının hesaplanması  
- Zaman serisi olarak günlük satış trendinin çıkarılması  
- Dashboard’ta KPI (Toplam Satış), kategori bazlı grafik, ürün bazlı grafik ve satış trendi görselleştirmeleri

## Nasıl İncelenir  
1. GitHub içerisindeki notebook dosyasını açarak kod adımlarını inceleyebilirsin.  
2. Tableau linkine tıklayarak interaktif dashboard’u görüntüleyebilirsin.  
3. Proje senin için benzer bir veride uygulanabilir: Veriyi yükle, temizle, analiz et, görselleştir.
